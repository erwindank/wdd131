Import from File

Upload a JSON or CSV file to import multiple entries at once.

Expected File Formats

JSON Format:

[
  {
    "song": "Song Title",
    "artist": "Artist Name",
    "album": "Album Name",
    "plays": 100,
    "datetime": "2025-01-01T14:30"
  }
]

CSV Format:

song,artist,album,plays,datetime
"Song Title","Artist Name","Album Name",100,"2025-01-01T14:30"
"Another Song","Another Artist","Album",50,"2025-01-02T10:00"

Note: All fields except album are required. The datetime should be in ISO format (YYYY-MM-DDTHH:MM).
